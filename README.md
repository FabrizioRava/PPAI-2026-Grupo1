# PPAI 2026 - Sistema de Seguimiento de Bolsines

## Grupo 1

Trabajo Práctico Aplicado Integrador de la cátedra Diseño de Sistemas de Información.

## Descripción

El sistema permite consultar la ubicación de bolsines en estado "Enviado" y generar una notificación con la ubicación actual dirigida al Gerente de la Comisión Médica de destino.

La implementación sigue el modelo de análisis desarrollado durante la materia, aplicando conceptos de orientación a objetos y patrones GRASP.
